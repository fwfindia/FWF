🆘 Need Help?

Common Solutions:

1. Mobile version not loading?
• Clear browser cache (Ctrl+Shift+Delete)
• Try incognito/private mode
• Check browser console for errors (F12)

2. Desktop version showing on mobile?
• Directly visit: yourdomain.com/m/
• Check if detection script loaded
• Test in Chrome DevTools mobile mode

3. Service worker not working?
• HTTPS required for PWA (works on localhost)
• Check Application tab in DevTools
• Unregister and re-register if needed

4. Images not loading?
• Check image paths are correct
• Verify lazy-load.js is included
• Test with different image sizes
